# Hostel-Utility-App

<h2 align="center">Email me at advitiya4673@gmail.com or advitiya9253@gmail.com for the source code of this project.</h2>

<p>Built a project named "Hostel Utility App" in the language JAVA and XML by using Firebase Database on Android Studio IDE. I used the Bottom-Up approach to build this application. It took me around 30 days to complete this work. I followed a YouTube channel named "Papaya Coders" from there I learned about Activity, Fragment, and all other stuff.

I started developing the app by making a list of features my app should have. Then I designed the app by taking reference from some Instagram pages. The Design is so beautiful, I used the "Flat Icon website" for icons and images. I also learned how to perform CRUD operations in the firebase database, upload and delete images in firebase storage, create users with the help of firebase authentication and how to write firebase queries in JAVA. I also included a Date Time API and used the Picasso Image Library. and also added the "sent password reset email via firebase" feature in my app.

At the start I had to watch YouTube videos to solve each error that came in my code, but as I went further, i developed many functions of the app without taking help from anywhere. I did everything, from designing the app to writing code, testing the app, and creating its documentation. And in the end, I have good experience in developing mobile applications.</p>
